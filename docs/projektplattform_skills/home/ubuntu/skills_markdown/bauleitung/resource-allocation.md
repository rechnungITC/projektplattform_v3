# Resource Allocation

## Übersicht
Dieser Skill optimiert die Zuteilung von Ressourcen, um einen reibungslosen Bauablauf ohne Stillstandzeiten zu gewährleisten.

## Anwendungsbereiche
- Personaleinsatzplanung
- Disposition von Baumaschinen
- Materialabruf und Logistikplanung

## Kernkompetenzen
- **Kapazitätsplanung**: Abgleich von benötigten und verfügbaren Ressourcen.
- **Konfliktlösung**: Vermeidung von Überbuchungen (Resource Leveling).
- **Logistik-Koordination**: Planung von Lieferfenstern (Just-in-Time).
- **Kostenüberwachung**: Tracking der Ressourcenkosten im Vergleich zum Budget.

## Workflow
1. Ressourcenbedarf aus dem Bauzeitenplan ableiten.
2. Verfügbarkeit von eigenem Personal und Maschinen prüfen.
3. Nachunternehmer und Mietgeräte rechtzeitig anfordern.
4. Materialabrufe gemäß Baufortschritt terminieren.
5. Tägliche Anpassung der Planung bei unvorhergesehenen Ereignissen (z.B. Wetter, Krankheit).

## Best Practices
- Engpassressourcen (z.B. Spezialmaschinen, Kranzeiten) frühzeitig blocken.
- Puffer für kritische Materialien einplanen.
