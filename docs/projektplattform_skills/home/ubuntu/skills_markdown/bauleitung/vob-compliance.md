# VOB Compliance

## Übersicht
Dieser Skill stellt sicher, dass Ausschreibungen, Vergaben und Abrechnungen konform mit der Vergabe- und Vertragsordnung für Bauleistungen (VOB) erfolgen.

## Anwendungsbereiche
- Erstellung von Leistungsverzeichnissen (LV)
- Angebotsprüfung
- Aufmaß- und Rechnungsprüfung

## Kernkompetenzen
- **VOB/A-Kenntnisse**: Richtlinien für die Vergabe von Bauaufträgen.
- **VOB/B-Kenntnisse**: Allgemeine Vertragsbedingungen für die Ausführung.
- **VOB/C-Kenntnisse**: Allgemeine Technische Vertragsbedingungen (ATV) für spezifische Gewerke.
- **Aufmaßprüfung**: Kontrolle der abgerechneten Mengen gemäß VOB-Regeln (z.B. Übermessungsregeln).

## Workflow
1. Leistungsverzeichnis auf VOB-Konformität prüfen.
2. Angebote formell und rechnerisch prüfen.
3. Aufmaße der Nachunternehmer gemäß VOB/C-Regeln kontrollieren.
4. Abschlags- und Schlussrechnungen freigeben.
5. Dokumentation für das Bautagebuch erstellen.

## Best Practices
- Bei Abweichungen von der VOB klare vertragliche Regelungen treffen.
- Aufmaße zeitnah und idealerweise gemeinsam mit dem Nachunternehmer prüfen.
