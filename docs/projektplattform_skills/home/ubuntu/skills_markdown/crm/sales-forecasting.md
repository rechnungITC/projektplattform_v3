# Sales Forecasting

## Übersicht
Dieser Skill bietet datengetriebene Vorhersagen über zukünftige Umsätze basierend auf der aktuellen Sales-Pipeline.

## Anwendungsbereiche
- Umsatzplanung
- Ressourcenallokation im Vertrieb
- Identifikation von Pipeline-Lücken

## Kernkompetenzen
- **Wahrscheinlichkeitsberechnung**: Zuordnung von Abschlusswahrscheinlichkeiten zu Sales-Stages.
- **Historische Analyse**: Auswertung vergangener Win/Loss-Raten.
- **Sales-Cycle-Tracking**: Messung der durchschnittlichen Dauer bis zum Abschluss.
- **Pipeline-Bereinigung**: Identifikation von "toten" Deals.

## Workflow
1. Verkaufsphasen (Sales Stages) klar definieren.
2. Historische Daten zur Kalibrierung der Wahrscheinlichkeiten nutzen.
3. Forecast-Modelle im CRM konfigurieren.
4. Regelmäßige Pipeline-Reviews mit dem Vertriebsteam durchführen.
5. Forecast-Berichte für das Management generieren.

## Best Practices
- Vertriebsmitarbeiter zur konsequenten Datenpflege motivieren.
- Best-Case und Worst-Case Szenarien abbilden.
