# Inventory Management

## Übersicht
Dieser Skill dient der Optimierung von Lagerbeständen, um Lieferfähigkeit bei minimaler Kapitalbindung zu gewährleisten.

## Anwendungsbereiche
- Automatische Disposition
- Reichweitenanalyse
- ABC/XYZ-Analyse
- Vermeidung von Out-of-Stock-Situationen

## Kernkompetenzen
- **Bedarfsprognose**: Nutzung historischer Daten und saisonaler Trends.
- **Meldebestandsberechnung**: Dynamische Anpassung von Sicherheits- und Meldebeständen.
- **Losgrößenoptimierung**: Berechnung der optimalen Bestellmenge (z.B. Andler-Formel).
- **Lieferantenbewertung**: Berücksichtigung von Lieferzeiten und -treue.

## Workflow
1. Analyse der historischen Verbrauchsdaten.
2. Klassifizierung der Artikel (ABC/XYZ).
3. Berechnung der Sicherheitsbestände basierend auf Servicegraden.
4. Generierung von Bestellvorschlägen bei Unterschreitung des Meldebestands.
5. Übergabe an den Einkauf.

## Best Practices
- Regelmäßige Überprüfung der Prognoseparameter.
- Berücksichtigung von Mindesthaltbarkeitsdaten bei verderblichen Waren.
