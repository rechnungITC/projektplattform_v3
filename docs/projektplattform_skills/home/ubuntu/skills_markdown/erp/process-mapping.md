# Process Mapping

## Übersicht
Dieser Skill standardisiert und visualisiert Geschäftsprozesse, um deren nahtlose Implementierung im ERP-System zu ermöglichen.

## Anwendungsbereiche
- Prozessanalyse und -optimierung
- Anforderungsspezifikation für ERP-Customizing
- Schulungsunterlagen für Endanwender

## Kernkompetenzen
- **BPMN 2.0**: Standardkonforme Modellierung von Prozessen.
- **Prozessanalyse**: Identifikation von Engpässen und Medienbrüchen.
- **Schnittstellendefinition**: Spezifikation von Systemgrenzen und Datenaustausch.
- **Rollenkonzept**: Zuordnung von Aufgaben zu Benutzerrollen (Swimlanes).

## Workflow
1. Interviews mit Fachabteilungen führen (Ist-Aufnahme).
2. Ist-Prozess in BPMN modellieren.
3. Schwachstellen analysieren und Soll-Prozess definieren.
4. Soll-Prozess mit ERP-Standardfunktionen abgleichen (Fit-Gap-Analyse).
5. Finale Prozessdokumentation erstellen.

## Best Practices
- Prozesse so detailliert wie nötig, so einfach wie möglich modellieren.
- Klare Definition von Start- und Endereignissen für jeden Prozess.
